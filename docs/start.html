Ivory: Downloading and setting up

Ivory uses Cloud9, a MapReduce library for Hadoop developed at the University of Maryland. It primarily started off as a teaching tool, but in the spirit of eating your own dog food, it is being actively used in several research projects.

Cloud9 is available as a tarball, or alternatively you can check out the latest code from our Subversion repository. Either way, you'll want to put both Cloud9 and Ivory in the same top-level directory. Specifically, you should have:

  • foo/umd-hadoop-core/ for Cloud9
  • foo/umd-hadoop-ivory/ for Ivory

Go into foo/umd-hadoop-ivory/ and type "ant". This should automatically build the distribution and create ivory.jar. The jar will contain all the class files in Cloud9 also, this jar is self-contained and ready for submitting to a Hadoop cluster.
