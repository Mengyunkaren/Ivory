Ivory: Third-Party Libraries

Ivory depends on the following third-party libraries, parts of which are included in the distribution:

  • Lucene for tokenization code.
  • Galgo for tokenization code (which itself uses the Snowball stemmer code from tartarus.org).
  • Terrier for compression code.

We gratefully acknowledge contributions from these above projects.
